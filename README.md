# Notes App

Notes App is a lightweight console application for managing personal notes.

The application groups notes into categories, supports searching by keyword and displays a formatted summary.

---

## Features

- Browse all notes
- Search by keyword
- Categories
- Simple console interface
- Sample data included

---

## Example

```
Notebook

Work

• Finish API documentation

Personal

• Buy groceries

Ideas

• Create a desktop weather widget

Total notes: 3
```

---

## Project Files

| File | Purpose |
|------|---------|
| app.py | Entry point |
| notebook.py | Note collection |
| repository.py | Data provider |
| search.py | Search functions |
| printer.py | Console output |
| sample_notes.py | Demo notes |

Run:

```bash
python app.py
```

No external database is required.
